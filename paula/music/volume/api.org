The volume module contains all functionality to handle the system sound volume

* Provided
** volume.py
  - get()
    Returns the systems sound volume.
  - set(percent)
    Sets the systems sound volume to "percent".
  - mute()
    Mutes the systems sound volume

* Required
** utils
*** shell_command
    - call(cmd)
    - call_silently(cmd)
